# Dev Journal Agent – "The Historian"

## Purpose
Auto-generate weekly dev journal entries from Git, VS Code, and Obsidian logs.

## Workflow
1. Gather:
   - Git commit logs
   - VS Code session notes
   - Obsidian `#devlog` notes
2. GPT formats into:
   - Timeline
   - Dev challenges
   - Lessons learned
   - Snippets w/ commentary

## Destination
- `docs.darlington.dev/journal/YYYY-MM-DD`
- Possibly also cross-posted to Hashnode for narrative pieces
