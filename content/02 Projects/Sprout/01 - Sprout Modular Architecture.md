# Sprout Modular Architecture

We mapped out a modular system of agents under the Sprout life OS concept:

## Core Agents
- **Sprout Core**: Work & focus tracker, deep work sessions, rituals
- **Ritualist**: Habits & distraction tracker (Oru integration, browser usage)
- **Fueler**: Diet reflection agent (Shortcut voice logs, mood-based analysis)
- **Mover**: Exercise reflection agent (not a planner, but insight generator)
- **Connector**: Social connection tracker (aggregates chats, nudges outreach)
- **Observer**: Meta-Coach for weekly review & theme generation
- **Synthesizer**: Learning agent (interview analysis, flashcard generation)
- **Historian**: Dev journal automation (Git, VS Code, Obsidian)

## Weekly Flow
- Daily tracking via voice memos + Shortcuts
- Friday: Observer pulls all agent logs, reflects on alignment
