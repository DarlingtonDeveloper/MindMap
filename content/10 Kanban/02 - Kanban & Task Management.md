# Kanban & Task Management Options

## Goal
Track tasks across agents, prioritize based on Impact / Momentum / Ease / Magic

## Best Options
- **Obsidian**: Local-first, Markdown-based, good for personal productivity
- **Planka**: Open-source Trello clone, can be hosted publicly (e.g. tasks.darlington.dev)
- **Focalboard**: Mattermost Boards alternative, also hostable
- **Quartz**: Ideal for public roadmap or static task board
- **Trello**: Easy embed in Quartz using iframe

## Recommended Flow
- Obsidian for daily notes and sprint logs
- Planka for public modular agent task board
- Quartz for publishing long-term roadmap
