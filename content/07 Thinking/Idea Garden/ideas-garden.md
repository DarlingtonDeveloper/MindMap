# 🌱 Ideas Garden

A space to plant, track, and grow project ideas.

---

## 🧰 DevTools Kit

**Problem**: Repeating tiny dev tasks (Googling regex, cleaning JSON, converting units).

**Idea**: Build microtools for each repetitive task and bundle them into a DevKit.

**Later**: Wrap into a CLI, VS Code extension, or web UI. Think “Swiss Army Knife for developers.”

---

## 🎓 YouTube Summarizer Extension

**Problem**: YouTube learning lacks structure. Current tools (e.g., Eightify) are limited or paid.

**Idea**: Chrome extension that offers:
- ✅ Summarize Transcript
- 🧠 Key Quotes
- 📚 Study Guide
- ❓ Q&A Generator
- 🧪 Quiz Mode
- ✨ Flash Cards + Highlights

**Later**: Integrate with Learning Agent or export to Obsidian/Markdown.

---

## 🌍 Remote Work Tracker

**Problem**: Applying for remote jobs is chaotic and hard to track.

**Idea**: Build a dashboard to:
- Find remote roles from curated sources
- Track saved/applied/interview status
- Sort by filters (e.g. async, Web3, GCP, etc.)

**Later**: Sync with Notion, Obsidian, or personal goal tracking tools.
