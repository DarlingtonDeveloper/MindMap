# 🌱 Weekly Launch Plan – Week 1

This week is about building a foundation for your 10-year life vision. Start small, go deep, and get aligned.

---

## 🧠 1. Reaffirm Your North Star

- [ ] Write or review your **life mission statement** in Obsidian
- [ ] Choose 2–3 “Feeling States” for the year (e.g. “Aligned, Energized, In Flow”)
- [ ] Revisit or write down your “Why” behind Sprout + Oru

📍Save to: `/_vision/core-values.md`

---

## 🧰 2. Set Up Weekly Review System

- [x] Create or update your `weekly-review.md` template
- [x] Include:
    - ✅ What energized me?
    - 🚫 What drained me?
    - 🎯 Am I aligned with my mission?
    - 🌱 Sprout Status (progress / habits)
- [x] Block time each weekend for “Observer Sync”

---

## 🔨 3. Launch MVP of Oru Logging

- [ ] Finalize Shortcut → Supabase `/api/log` connection
- [ ] Log 3+ distraction events manually
- [ ] Create a test dashboard or CSV output
- [ ] Journal the triggers of each scroll

---

## 🌿 4. Use Sprout for 3 Focus Sessions

- [ ] Define task, goal, timer for each session
- [ ] Reflect after: Did you enter flow? Did the metaphor help?
- [ ] Log results in `sprout-log.md`

---

## 💬 5. Share Publicly Once

- [ ] Post a tweet/thread about what you’re building
- [ ] Optional: share it in an indie hacker or dev tools Discord
- [ ] Bonus: get early feedback

---

## 🧱 6. Set Up `darlington.dev/docs/vision`

- [ ] Create `docs/vision/index.md`
- [ ] Add links to:
    - `10-Year Trajectory`
    - `Observer Agent`
    - `Sprout Log`
    - `Roadmap.md`
- [ ] Test local build or push to Vercel

---

## 🌞 7. Do 1 Joy-First Activity

- [ ] Walk, swim, play, or create with zero output expectation
- [ ] Journal: *“What did I love as a kid that I forgot?”*

---

## 📅 Suggested Schedule

| Day   | Focus                                 |
|--------|----------------------------------------|
| Mon   | Clarity (Tasks 1 & 2)                  |
| Tue   | Oru Logging (Task 3)                   |
| Wed   | Sprout Session #1 (Task 4)             |
| Thu   | Work + Public Share (Task 5)           |
| Fri   | Vision Page Setup (Task 6)             |
| Sat   | Observer Sync Review                   |
| Sun   | Joy-First Reset (Task 7)               |

---

*This plan is designed to be reusable each time you start a new system phase.*
